User Guide

This guide will walk you through the features of the Course Scheduler.

1. Dashboard

The dashboard includes:

  • A calendar or list view displaying current and upcoming events.
  • A navigation bar with the different pages: Edit, View, and Preferences.
  • View: Shows the generated schedule and download options.
  • Edit: Change teachers, courses, and classrooms.

2. View Page

The view page allows you to:

  • Download data as JSON or CSV (lower right hand corner)
  • Generate a schedule based on the data from the edit page.

3. Edit Page

The edit page includes three columns:

  • Teachers: Edit teacher details and courses taught.
  • Courses: Edit course details, compatible classrooms, and periods.
  • Classrooms: Edit classroom availability.

4. Edit Page-View Page Interaction
